# Onzio Prospect Template Product Rules

- One repository and Vercel deployment per prospect
- Master remains fictional and reusable
- Generated previews are independent snapshots
- Pro opens by default; Starter remains available for comparison
- Same premium layout across prospects
- Real club identity, facts, palette, crest, photos, links, and merchandise imagery
- Fictional sample roster, staff, fixtures, results, statistics, and sponsors are allowed
- Codex drafts identity-driven copy; Christian approves it
- Original assets are preserved outside generated repositories
- Optimized copies may be resized, compressed, cropped, background-removed, renamed, and given alt text
- Public GitHub repositories are temporarily acceptable; include no secrets or private information
- `noindex` and `nofollow` remain mandatory
- Generated previews are sales artifacts, not production systems
- Nothing is published without Christian's explicit green light
